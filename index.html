Domain name/Hosting

01.
Hiểu về
DNS.
Hệ thống phân giải tên miền (hay được viết tắt là DNS do tên tiếng Anh Domain Name System) là một hệ thống cho phép thiết lập tương ứng giữa địa chỉ IP và tên miền trên Internet.
02.
Đăng ký
Tên miền miễn phí.
Đăng ký tên miền để Xây dựng và bảo vệ thương hiệu, xây dựng website để quản lý, dạy học, làm việc, làm thương mại, quảng cáo, quảng bá thương hiệu.
03.
Đăng ký
Tên miền có phí.
Tên miền chính là thương hiệu trên thế giới ảo của bạn. Tên miễn trả phí giúp mọi người dễ nhớ hơn và có cảm tình hơn.
04.
Hiểu về
Web server.
Web server dịch ra tiếng Việt nghĩa là máy chủ. Web server là máy tính lớn được kết nối với tập hợp mạng máy tính mở rộng.
05.
Cài đặt và cấu hình
Xampp/Wamp.
Wamp và Xampp là hai phần mềm giả lập server trên localhost cực kì tiện lợi, chúng có cả MySQL, php và Apache được tích hợp cùng và có hàng tá các tiện ích khác.
06.
Tìm hiểu
Về các loại hosting.
Hosting (hay web hosting) là một dịch vụ online giúp bạn xuất bản website hoặc ứng dụng web lên Internet.
07.
Thông số liên quan
Đến một web hosting.
Tìm hiểu các thông số lên quan đến một web hosting như dung lượng lưu trữ/trữ, băng thông, cơ chế bảo mật, sao lưu phục hồi dữ liệu, ...
08.
Shared hosting
Miễn phí.
Thực hiện đăng kí tên miền và chỗ đặt website miễn phí.Tự tạo một website đơn giản (có thể chỉ gồm một trang index.html với nội dung là một đoạn văn bản).
09.
Shared hosting
Có phí.
“Shared hosting trả phí” có nhiều thành phần và dịch vụ để cấu hình hơn so với “shared hosting miễn phí”. Tuy nhiên, cách đưa một website lên hosting thì tương tự.
